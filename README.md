# 📅 CampusMate - 學霸課表

**CampusMate** 是一個專為學生設計的現代化課表與成績管理工具。結合 PWA 技術，讓您能直接安裝於手機桌面，隨時隨地輕鬆管理學習生活。

## ✨ 主要特色

### 1. 📝 智慧課表管理
* **每週總覽**：直覺式的週一至週五分頁切換。
* **詳細資訊**：記錄節次、時間、科目、教室位置及老師。

### 2. 📊 雙模式成績試算
* **🎓 大學生模式**：支援學分輸入，自動計算加權 GPA。
* **🏫 高中生模式**：專注於分數，自動計算算術平均。

### 3. 🗓️ 多學期檔案系統
* **完整記錄**：支援建立多個學期檔案（如 113-1, 113-2）。

### 4. ☁️ 帳號
* **集中管理**：帳號登出與身分切換功能已整合至「設定」頁面。

### 5. 📱 PWA 支援
* **安裝至桌面**：支援 Add to Home Screen，提供如同原生 App 的體驗。

### 6. JavaScript檔案功能
* ** js / firebase.js **(設定檔)：負責 Firebase 的初始化與 API 設定。
* ** js / state.js **(全域變數)：負責管理所有的共用變數，讓其他檔案可以讀取。
* ** js / ui.js **(介面控制)：負責頁面切換、分頁控制等通用 UI 功能。
* ** js / course.js **(課程管理)：負責課表的顯示、新增、刪除邏輯。
* ** js / grade.js **(成績管理)：負責成績的顯示、計算、新增、刪除邏輯。
* ** js / data.js **(資料核心)：負責存檔 (saveData)、讀檔 (loadData) 與舊資料遷移。
* ** js / semester.js **(學期管理)：負責學期的新增、切換、刪除。
* ** js / auth.js **(帳號管理)：負責登入、登出、註冊與身分切換。
* ** js / main.js **(程式入口)：負責整合並啟動程式，監聽登入狀態。
---

## 📖 使用說明
1. **初次設定**：登入後請先選擇身分（高中/大學），以設定計算公式。
2. **新增課程**：在首頁點擊「+ 編輯本日課程」即可輸入。
3. **帳號管理**：如需切換身分或登出，請前往底部導覽列的「⚙️ 設定」頁面。
