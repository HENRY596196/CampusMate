# 📅 CampusMate - 學生課表小幫手

**CampusMate** 是一個專為高中生與大學生設計的課表管理 App。它採用 PWA (Progressive Web App) 技術，可以像原生 App 一樣安裝在手機上，並結合 **Google Firebase** 實現帳號登入與資料同步功能。

## ✨ 特色功能

* **👥 雙重身分模式**：
    * **大學生**：支援學分顯示、GPA 自動計算。
    * **高中生**：簡化介面，專注於課程與成績。
* **📅 週課表管理**：
    * 提供週一至週五的分頁切換。
    * 自動偵測並顯示當天的課程。
    * 支援新增、編輯與刪除課程。
* **☁️ 雲端身分驗證 (Firebase)**：
    * 支援 **Google 一鍵登入**。
    * 資料依據使用者帳號 (UID) 隔離儲存，多人共用裝置也不會搞混資料。
* **📱 PWA 支援**：
    * 支援離線瀏覽。
    * 可透過瀏覽器「加入主畫面」安裝成 App。
    * 如同原生 App 的操作體驗 (無網址列)。

## 🛠️ 技術架構

* **前端核心**：HTML5, CSS3, Vanilla JavaScript (無須編譯/打包)
* **後端服務**：Google Firebase (Authentication)
* **資料儲存**：LocalStorage + Firebase UID (資料隔離)
* **部署平台**：GitHub Pages / Vercel

```text
CampusMate/

```

成績單功能亮點
自動判斷計算方式：

*如果你是大學生：系統會自動計算 「加權平均」 (分數 × 學分 ÷ 總學分)，輸入視窗也會有「學分」欄位。
*如果你是高中生：系統會計算 「算術平均」 (總分 ÷ 科目數)，且輸入視窗會自動隱藏「學分」欄位。
*資料同步：成績單現在也會跟著你的 Google 帳號儲存了！
*紅綠燈顯示：不及格 (低於 60 分) 的成績會自動變成紅色，及格則是綠色。
