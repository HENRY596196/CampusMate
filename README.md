# 📅 CampusMate - 學霸課表

**CampusMate** 是一個專為學生設計的現代化課表與成績管理工具。結合 PWA 技術，讓您能直接安裝於手機桌面，隨時隨地輕鬆管理學習生活。

## ✨ 主要特色

### 1. 📝 智慧課表管理
* **每週總覽**：直覺式的週一至週五分頁切換。
* **詳細資訊**：記錄節次、時間、科目、教室位置及老師。

### 2. 📊 雙模式成績試算
* **🎓 大學生模式**：支援學分輸入，自動計算加權 GPA。
* **🏫 高中生模式**：專注於分數，自動計算算術平均。

### 3. 🗓️ 多學期檔案系統
* **完整記錄**：支援建立多個學期檔案（如 113-1, 113-2）。

### 4. ☁️ 雲端同步與帳號
* **Firebase 同步**：資料安全儲存於雲端，跨裝置自動同步。
* **集中管理**：帳號登出與身分切換功能已整合至「設定」頁面。

### 5. 📱 PWA 支援
* **安裝至桌面**：支援 Add to Home Screen，提供如同原生 App 的體驗。

---

## 📖 使用說明
1. **初次設定**：登入後請先選擇身分（高中/大學），以設定計算公式。
2. **新增課程**：在首頁點擊「+ 編輯本日課程」即可輸入。
3. **帳號管理**：如需切換身分或登出，請前往底部導覽列的「⚙️ 設定」頁面。
